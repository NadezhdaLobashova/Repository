Документ предназначен для фиксирования плановых отсутствий сотрудника. Факт отсутствия сотрудника фиксируется документом ЛУРВ.

Дата начала - день, когда отсутствует сотрудник.

Вид отсутствия - указывается из справочника "Работы".

Часы - часы отсутствия сотрудника в день (не должно превышать 8 часов).

На форме документа предусмотрена форма заполнения табличной части по периоду. Табличная часть документа заполняется рабочими днями из периода, указанного на форме, согласно производственному календарю. Количество часов на форме не должно превышать 8ч.
