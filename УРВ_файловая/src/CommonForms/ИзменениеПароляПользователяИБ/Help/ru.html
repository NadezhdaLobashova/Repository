Для изменения пароля к информационной базе введите новый пароль и нажмите Установить пароль.

Чтобы видеть вводимый пароль, установите флажок Показать пароль.

Для создания безопасного пароля из заглавных и строчных букв, а также цифр и специальных символов, воспользуйтесь кнопкой Случайный пароль.

Если выводится сообщение "Заданный　пароль　не　удовлетворяет　требованиям　сложности", это означает, что администратор программы разрешил использовать только сложные пароли:

  • длина не менее 7 символов,
  • в пароле используются заглавные, строчные буквы, цифры, специальные символы,
  • не совпадает с именем пользователя,
  • не является последовательностью символов, например "12345", "qwerty".
Когда пароль изменить нельзя, программа не станет открывать окно смены пароля, а покажет одно из следующих предупреждений:
  • Текущий пользователь не определен. Изменение пароля невозможно. Обратитесь к администратору. 
    Вход выполнен анонимно, без указания пользователя. Такая ситуация может встречаться при первом входе в программу.
  • Изменение пароля не требуется, т.к. используется аутентификация операционной системы.
    В карточке пользователя снят флажок Аутентификация 1С:Предприятия, и настроена аутентификация операционной системы.
  • Изменение пароля запрещено. Обратитесь к администратору.
    В карточке пользователя установлен флаг Пользователю запрещено изменять пароль.
