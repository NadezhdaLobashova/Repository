Документ предназначен для учета прогноза по проекту.

Сценарий - устанавливается значение "Прогноз".

Период прогноза - период действия прогноза по проекту.

Табличная часть:

Период - период планирования по проекту.

Статья - статья планирования.

Подразделение сотрудника - подразделение, в котором работает сотрудник на дату документа.

Должность - должность сотрудника на дату документа.

Контрагент - указывается, если в проекте установлено значение "Учет по контрагентам".

Оклад/Ставка - оклад/ставка сотрудника на дату документа.

Часы - план часов на сотрудника.

 На форме документа предусмотрен подбор сотрудников по подразделениям, видам работ и ставкам - кнопка "Подбор".
