Документ служит для учета факта работы сотрудника.

Документ заполняется ежедневно.

В табличной части указывается:

1. Вид работы. Вид работы включается в себя оплачиваюмую работу, отгулы, отпрашивания, отпуска и т.п.

2. Задача. Указывается при наличие назначенной сотруднику задачи.

3. Проект. Проект, по которому выполняется работа. В случае, когда указывается регламентированная работа проект заполняется обязательно. 

4. Контрагент. Указывается для проектов, у которых установлен признак  "Учет по контрагентам".

5. Выходной. Устанавливается, если работа выполнялась в выходной/праздничный день и согласована с руководством.
